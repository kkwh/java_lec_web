HTTP

비연결(non-connection) 지향성 프로토콜, 무상태(stateless) 프로토콜
클라이언트(웹 브라우저)의 요청(request)에 대한 응답(response)을 하고 나면,
서버는 해당 클라이언트와의 연결을 지속하지 않고 끊어버림

쿠키(Cookie)
(필요한 정보를 클라이언트(웹 브라우저)에서 저장)
웹 브라우저의 최초 요청 시 웹 서버는 쿠키를 생성해서
브라우저로 전송
웹 브라우저는 쿠키를 저장
웹 브라우저가 같은 서버에 접속할 때는 저장된 쿠키를 요청에
실어서 보냄
- 보안에 취약
- 언제든지 쿠키를 지울 수 있음

세션(Session)
(서버에서 저장)
